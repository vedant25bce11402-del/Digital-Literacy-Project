# Project Overview

This project is part of the Digital Literacy (CSE0001) course, where I have taken on the role of a Student Digital Ambassador. The goal of this project is to build awareness among students about effective, safe, and responsible use of digital tools and platforms.
The project is organized as a Digital Literacy Portfolio, covering five key modules including online presence, communication, collaboration tools, and cyber safety.


## 🧩 Module Summary

### 🔹 Task 1: Digital Literacy Infographic

Created an awareness infographic using a design tool to explain:

* What digital literacy is
* Safe internet practices
* Importance of professional online presence

---

### 🔹 Task 2: Digital Portfolio

Set up professional profiles on platforms like:

* GitHub
* LinkedIn
* Other relevant platforms

This helps in building a strong online presence for future academic and career opportunities.


### 🔹 Task 3: Platforms Exploration

* Completed a beginner coding challenge on a coding platform
* Created a **Google Form quiz** on digital literacy
* Learned how collaboration tools help in academics


### 🔹 Task 4: Email Etiquette & Social Media

* Drafted professional emails:

  * Request for assignment extension
  * Internship application
* Created a checklist of **Social Media Do’s and Don’ts**


### 🔹 Task 5: Cybercrime Awareness

* Wrote a case study on a cybercrime topic (e.g., phishing)
* Created a **“Stay Safe Online” checklist** including UPI safety tips
* Included reporting methods:

  * National Cyber Crime Portal: https://cybercrime.gov.in
  * Helpline: **1930**
